# Age-Calculator-🎂 

A simple and accurate **Age Calculator App** built with **HTML**, **CSS**, and **JavaScript**.  
This app allows users to select their date of birth and instantly calculates their age in **years**, **months**, and **days**.

Perfect for beginners learning date manipulation, JavaScript logic, and interactive UI design.



## 🚀 Features

### 🔹 Core Features
- Select date of birth  
- Instant age calculation  
- Shows:
  - Age in years  
  - Age in months  
  - Age in days  

### 🎨 UI Features
- Clean and modern interface  
- Responsive layout  
- Easy-to-use date input  


## 📂 Project Structure



age-calculator/ │ ├── index.html ├── style.css └── app.js



## 💻 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – Age calculation logic  

No frameworks required
